# 中文版本： 社交媒体成瘾分析项目  English follows

## 📌 项目简介
本项目旨在分析社交媒体使用对学生学业表现、睡眠状况以及心理健康的影响。通过数据分析方法，探究过度使用社交媒体所带来的潜在风险。

## 🎯 项目目标
- 分析学生社交媒体使用的总体情况
- 比较不同性别和学业阶段的使用差异
- 研究使用时长与睡眠、心理健康之间的关系
- 探索是否存在“临界使用时间”，即超过某个使用时长后负面影响显著增加

## 📊 数据说明
数据集包含以下信息：
- 每日社交媒体使用时长
- 成瘾评分
- 睡眠时长
- 学业表现指标
- 学生基本信息（如性别、年级等）

## ⚙️ 分析方法
- 描述性统计分析
- 分组比较分析
- 相关性分析
- 阈值（临界点）分析

## 💻 使用工具
- Python（Pandas、NumPy、Matplotlib）
- Jupyter Notebook
